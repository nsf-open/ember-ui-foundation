@nsf/ui-foundation
==============================================================================

The starting point for Ember Application UI libraries at NSF.


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.8 or above
* Ember CLI v3.20 or above
* Node.js v12 or above


Installation
------------------------------------------------------------------------------

```
ember install @nsf/ui-foundation
```
